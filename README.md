# NeuronGraph #
The implementation of neuron network training in the form of graph.
***
### Features
* Apply the graph to simplify the code.
* Implement multithread computations.
* OOP coding.
* Features from [NeuronS](https://github.com/AdamQinwt/NeuronS).
